MDRO患者分布地图 - 使用说明
=====================================

📁 文件结构
-----------
MDRO_Map_App/
├── index.html              # 主地图应用
├── data_processor.py       # 数据处理器
├── data/                   # 数据文件夹
│   ├── mdro_patients.json  # 患者数据
│   ├── heatmap_data.json   # 热力图数据
│   └── data_stats.json     # 统计信息
├── README_MDRO_Map.md      # 详细说明文档
├── 更新说明.md             # 更新记录
├── 启动地图.bat            # 启动脚本
└── 使用说明.txt            # 本文件

🚀 快速启动
-----------
方法1：双击"启动地图.bat"文件（推荐）
方法2：命令行运行：
   1. 打开命令提示符
   2. 进入MDRO_Map_App文件夹
   3. 运行：python -m http.server 8000
   4. 浏览器访问：http://localhost:8000

⚠️  重要提示
-----------
- 直接双击index.html文件无法正常显示地图
- 必须通过HTTP服务器访问（方法1或方法2）
- 系统会自动检测访问方式并给出相应提示

📊 功能说明
-----------
1. 患者聚类：默认显示，深红色圆圈表示患者聚集
2. 患者点图：显示具体患者位置，不同菌种用不同颜色
3. 热力图：显示患者密度分布
4. 菌种筛选：可按ESBLE、MRSA、CRO筛选
5. 底图切换：简洁地图、深色地图、街道地图

🎨 界面布局
-----------
- 左上角：显示控制面板
- 右上角：统计信息面板
- 左下角：图层控制面板
- 右下角：图例说明面板

📈 数据概览
-----------
- 总患者数：454例
- 菌种分布：ESBLE(404例) > MRSA(40例) > CRO(10例)
- 地理范围：广东省及周边地区

🔧 数据处理
-----------
如需更新数据：
1. 将新的MDRO.xlsx文件放在此文件夹
2. 运行：python data_processor.py
3. 刷新浏览器页面

📞 技术支持
-----------
如有问题，请查看README_MDRO_Map.md获取详细技术说明。
